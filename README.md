# weather-forecast-app
Interactive weather app that shows a 5-day forecast, maps, and data visualizations using the OpenWeatherMap API

## Features
- **Current Weather**: Real-time weather conditions for any searchable city
- **5-Day Forecast**: Detailed daily forecasts with temperature trends
- **Interactive Maps**: Visualize weather patterns with map overlay
- **Responsive Design**: Works on both desktop and mobile

## Technologies being Used (update if anything new changes if not, FIXME and delete)

- HTML5, CSS3, Javascript (ES6+)
- OpenWeatherMap API
- Leaflet.js (Maps)
- Chart.js (data visualization)

## Installation

1. Close the repository:
```bash
git clone git@github.com:Yariee/weather-forecast-app.git
cd weather-forecast-app
```
2. Obtain and free API key from [OpenWeatherApp](https://openweathermap.org/api)

3. Create 'js/config.js':
```bash
cd js/config.js
```

4. Add your API key to 'js/config.js'

5. Open 'index.html' in your browser or local server
